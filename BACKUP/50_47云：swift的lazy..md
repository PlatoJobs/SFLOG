# [47云：swift的lazy ](https://github.com/platojobs/SFLOG/issues/50)

+ `lazy` 修饰符，`lazy属性`
+ `lazy` 方法， 集合的一个方法

>` lazy属性`就是初始值直到第一次使用的时候才执行计算的属性
注意：lazy属性必须是变量（`var修饰符`），因为常量属性（let修饰符）必须在初始化之前就有值，所以常量属性不能定义为`lazy`。

**延迟加载主要有以下两个使用的场景**：
1. 属性的初始值依赖于其他的属性值，只有其他的属性值有值之后才能得出该属性的值。
2. 属性的初始值需要大量的计算。
